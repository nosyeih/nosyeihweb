01 // El Desafío

La institución financiera necesitaba una plataforma capaz de procesar más de 100,000 transacciones por segundo y visualizarlas en tiempo real para sus gestores de riesgo. El sistema anterior heredado provocaba bloqueos del navegador y la latencia de datos era superior a los 5 minutos.

02 // La Solución Arquitectónica

Diseñé y desarrollé una Single Page Application utilizando React y WebSockets para mantener un flujo bidireccional constante. En el backend, Node.js se encargó de la orquestación junto con Redis para el almacenamiento en caché rápido, disminuyendo la latencia de la base de datos MongoDB al mínimo operacional.

  • Reducción de latencia del 40%
  • Soporte de +10k usuarios simultáneos sin degradación
  • Cobertura de tests E2E del 95%
